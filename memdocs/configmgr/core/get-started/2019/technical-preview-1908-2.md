# <a name="features-in-configuration-manager-technical-preview-version-19082"></a>Features in Technical Preview-Version 1908.2 für Configuration Manager

*Gilt für: Configuration Manager (Technical Preview-Branch)*

In diesem Artikel werden die Features vorgestellt, die in der Technical Preview-Version 1908.2 für Configuration Manager verfügbar sind. Installieren Sie diese Version, um Features für Ihren Technical Preview-Standort zu aktualisieren oder neue Features hinzuzufügen.
