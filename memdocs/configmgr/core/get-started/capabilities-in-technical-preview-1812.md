# <a name="capabilities-in-configuration-manager-technical-preview-version-1812"></a>Funktionen in der Technical Preview-Version 1812 für Configuration Manager 

*Gilt für: Configuration Manager (Technical Preview-Branch)*

In diesem Artikel werden die Features vorgestellt, die in der Technical Preview-Version 1812 für Configuration Manager verfügbar sind. Installieren Sie diese Version, um Features für Ihren Technical Preview-Standort zu aktualisieren oder neue Features hinzuzufügen. 

Lesen Sie den [Technical Preview](technical-preview.md)-Artikel vor der Installation dieses Updates. In diesem Artikel erhalten Sie Informationen zu den allgemeinen Anforderungen und Einschränkungen, die für die Verwendung der Technical Preview gelten. Außerdem erfahren Sie, wie Sie Updates zwischen den Versionen ausführen und Feedback übermitteln.
