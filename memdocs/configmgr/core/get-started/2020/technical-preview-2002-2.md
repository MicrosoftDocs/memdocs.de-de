# <a name="features-in-configuration-manager-technical-preview-version-20022"></a>Features in der Technical Preview-Version 2002.2 von Configuration Manager

*Gilt für: Configuration Manager (Technical Preview-Branch)*

In diesem Artikel werden die Features vorgestellt, die in der Technical Preview-Version 2002.2 von Configuration Manager verfügbar sind. Installieren Sie diese Version, um Features für Ihren Technical Preview-Standort zu aktualisieren oder neue Features hinzuzufügen.

Lesen Sie den [Technical Preview](../technical-preview.md)-Artikel vor der Installation dieses Updates. In diesem Artikel erhalten Sie Informationen zu den allgemeinen Anforderungen und Einschränkungen, die für die Verwendung der Technical Preview gelten. Außerdem erfahren Sie, wie Sie Updates zwischen den Versionen ausführen und Feedback übermitteln.
